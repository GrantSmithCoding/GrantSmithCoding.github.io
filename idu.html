Overview

This was the project that we did for the Informatics Capstone for the Informtics major at IU. This project was a part of earning the Informatics degree at IU.

Concept

Our concept is Identify U! It is a platform where underrepresented groups can connect and network!

Features:

  • Create posts
  • Add comments to posts
  • See events on a calander
  • See events on the map
  • Change languages
  • Login with IU login

Wireframes

We used Figma to create wireframes that would help us develop the flow of the site and so we know what we are doing going in.
